# Change Log

## [1.5.0]

### Added
- Test runner for example files (`npm test`, `npm run test:file`)
- Support for `null` literal (empty list)
- Support for `or` and `and` logical operators
- Support for boolean literals `#t` (true) and `#f` (false)
- Extension icon

### Changed
- Updated VSCode engine requirement to `^1.107.0`
- Translated all UI strings and documentation to English for marketplace release
- Added `activationEvents` for proper extension activation

### Fixed
- Fixed issues with boolean literals in conditional expressions
- Fixed `null` handling in list operations

## [1.0.0]

- Initial release
